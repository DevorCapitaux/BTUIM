## Getting Started

### Prerequisites

To run BTUIM you need:

- ncurses
- jp2a
- wget
- sysstat
- sox

### 1. Clone the repository

```shell
git clone https://github.com/DevorCapitaux/BTUIM.git
cd BTUIM
```

### 2. Run

```shell
./main
```
